# 1sem
Shorten long videos to one second every minute (1sem).

# Usage
```
php -f 1sem.php video.file
```
output filename is video.1sem.file

# Requirements
- PHP (created and tested on 8.3.0RC4)
- [FFMpeg and FFProbe](https://ffmpeg.org/)
